# Project Name

A calculator web app made with React using create-react-app

# Built With

- Major Languages
  - HTML
  - SASS
  - JavaScript

- Initializer
  - create-react-app

## Getting Started

- Clone this repo
- Install dependencies 
```
npm install
```
- Run build script
```
npm run build
```
- You can find the built web app on ./dist
